About Settings

Everything that should already be in the game... and more

You can fully customize your experience in Settins

If you have any questions head to Guide

Extension made by I am Shrek - Kapi73#3815

Features:

  • Chat
    • interactive invites with ship preview
    • built-in translator
    • commands auto-fill
    • click player name to manage him
    • often used phrases
    • smooth animations
  • Buttons
    • new buttons: rejoin, instant save, invite
    • customizable buttons
  • User Interface
    • modern game menu
    • customizable menu colors
    • animations
    • customizable scrollbars
  • Managing ship
    • crew amount statistics
    • player list ship under TAB key
  • QOL - Quality of Life
    • hotkeys for crucial tasks - exit, gravity, save etc.
    • toggle ' (cinematic mode) and / (stats)
    • favourite ships
    • snakecopter!
    • quick Texture pack swap
    • emergency distress ping for discord

Help

  • As captain click on user name to manage player status
  • Make sure to check Settings to don't miss any feature!
  • You can specify hotkey (Ctrl or Shift) and key for hotkeys
  • To make texture swap work, zip has to be correctly built
  • To translate someone's messages double click on it (non-system messages)
  • To translate your messages enter code in chat and press CTRL+ENTER or click the Globe
